# Eval Cases

This directory will hold design-to-plan-owned eval cases in follow-up PRs. Each case owns its seam
fixtures, candidate plan or review output, expected fixture data, rubric, bad-candidate examples,
and manifest.

This bootstrap PR intentionally does not add semantic Planning cases. The first case batch should
cover projection-only planning, missing-input stop behavior, source-closure defects, evidence
binding, and prevention of Jig schema invention.
