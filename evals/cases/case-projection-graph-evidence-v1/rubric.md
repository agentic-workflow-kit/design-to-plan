# Rubric

A passing candidate emits a plan that stays projection-only, preserves Jig plan properties by name,
co-locates each story with Product AC IDs, handoff fact IDs, and Jig properties, closes every
producer/consumer relation, cites source-backed dependency edges, and binds evidence to concrete
commands, checks, or artifacts. It must not invent a Jig schema, validator, runtime package, package
layout, or TypeScript interface.
